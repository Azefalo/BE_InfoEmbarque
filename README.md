### Used pins:

**PA0**:    EncoderA

**PA1**:    EncoderB

PA2:    NOT USED

PA3:    NOT USED

**PA4**:    Encoder index (Z)

PA5:    NOT USED

**PA6**:    Sens du Plateau

**PA7**:    Battery level
    
**PA9**:    RX UART

**PA10**:   TX UART

PA13:    NOT USED

PA14:    NOT USED

PA15:    NOT USED

**PB0**:    PWM ServoMotor


**PB1**:    Roulis

**PB2**:    Roulis

**PB5**:    Roulis

**PB6**:    Roulis
